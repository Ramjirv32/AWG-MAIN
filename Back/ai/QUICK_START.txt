📊 ML MODEL ARCHITECTURE
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

MODEL 1: FLOW PREDICTION
─────────────────────────
Input:       Temperature, Humidity, Water Level
Processing:  Dense(32) → Dense(16) → Dense(8)
Output:      Flow rate (0-500 ml/hour)
Latency:     ~2ms


MODEL 2: SILICA ESTIMATION
───────────────────────────
Input:       RGB sensor (Red, Green, Blue)
Processing:  Dense(16) → Dense(8)
Output:      Silica % (0-100%)
Latency:     ~2ms


MODEL 3: PRODUCTION DECISION
─────────────────────────────
Input:       Temperature, Humidity, Silica %
Processing:  Dense(16) → Dense(8) → Sigmoid
Output:      Should produce? (0.0-1.0)
Latency:     ~1ms


TOTAL INFERENCE TIME: ~5ms per cycle (every 5 seconds)

  ⚠️  TDS >1000 PPM (contaminated water)

  ⚠️  System errors detected
  ⚠️  Manual override disabled AND AI says "WAIT"
